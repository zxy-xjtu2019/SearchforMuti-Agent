#### 要素
+ 两个以上参与者
+ 利益
+ 策略
+ 信息

#### 分类
+ 根据是否能达到最终一致的协议
  + 合作博弈
  + 非合作博弈
+ 根据行动是否有顺序
  + 静态博弈
  + 动态博弈
+ 根据信息掌握的程度
  + 完全信息博弈
  + 不完全信息博弈
+ 根据最终受益情况
  + 负和博弈
  + 零和博弈
  + 正和博弈

#### 纳什均衡
+ 针对非合作博弈
+ 针对一件事情，不同的参与者会做出针对自身最优的策略，在不断变化中，这个策略最终会达到一种平衡
> 根据最终策略所带来的收益可以分为好平衡坏平衡

+ 个体最优决策不一定导致群体最优
+ 遵循规则的合作往往造就好平衡
+ 利他主义往往是群体最优解
